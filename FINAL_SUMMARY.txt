================================================================================
                              HOW TO USE IT
================================================================================

BUILD:
  make clean && make

RUN:
  ./run.sh
  
  OR
  
  qemu-system-i386 -fda build/main_floppy.img -boot a -m 32M -nic user,model=ne2000

WHAT YOU'LL SEE:
  1. OS boots from floppy
  2. Kernel loads
  3. Network driver initializes (NE2000)
  4. Browser starts automatically
  5. Demo pages display with rendered HTML
  6. System halts gracefully
